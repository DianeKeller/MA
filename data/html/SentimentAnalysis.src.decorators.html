Notes

This function never returns False as it raises an exception if the check of the attribute chain fails.

Examples

...     def __init__(self):

...     def __init__(self):
...         self.b = C()

...         self.a = B()

>>> obj = MyObject()

...     'Attribute chain is not valid'

  • self (Any) – The instance from which the log is triggered.

  • msg (str) – The message to be logged.

  • *args (Tuple[Any, ...]) – Additional arguments that may include an object with its own log method.

  • If self has a log method, it is used.

  • If neither condition is met, the module-level logger is used.
